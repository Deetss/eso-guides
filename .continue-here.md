## 📌 Current Position
We built and pre-configured a complete **Local Addon Sync system** that connects active in-game character stats, CP levels, active Mundus, and slotted gear traits directly to the web dashboard:
1. **EsoCompanion Addon**: A lightweight cross-platform Lua addon that tracks player character stats and writes them to local SavedVariables.
2. **Tkinter GUI Sync App (`companion_app.py`)**: A Windows-ready standalone GUI utility that lets non-technical users browse for their ESO directory, copy/install the addon on button click, watch for in-game state changes, and serve a CORS-enabled HTTP server at `http://localhost:8000/profile` to sync with the live hosted website.
3. **Web Dashboard Integration**: Dynamically checks for local loopback CORS service (localhost:8000) or relative `live_profile.json` on page load, automatically applying your class selector, CP level slider, active Mundus Stone, and equipped traits.
