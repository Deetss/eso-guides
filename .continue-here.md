## 📌 Current Position
We built and pre-configured a complete **Local Addon Sync system** that connects active in-game character stats, CP levels, active Mundus, and slotted gear traits directly to the web dashboard:
1. **EsoCompanion Addon**: A lightweight cross-platform Lua addon that tracks player character stats and writes them to local SavedVariables.
2. **Local Sync Client (`sync.js`)**: A Node.js background file watcher that monitors the `EsoCompanion.lua` SavedVariables file, parses it, and writes `live_profile.json` into the public website directories.
3. **Web Dashboard Integration**: Dynamically fetches `live_profile.json` on page load, automatically applying your class selector, CP level slider, active Mundus Stone, and equipped traits.

## 🚀 Resume Instructions
To run locally:
```bash
npm run dev
```
To run the in-game sync client:
1. Configure your Documents path in `config.json` (defaults to standard Windows/Linux folder).
2. Install the addon: `node scripts/install-addon.js`.
3. Launch the sync client: `node scripts/sync.js`.
4. Log out or reload UI in-game to see the web app update instantly!
